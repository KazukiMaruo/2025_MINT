# MINT


> ❓ Research question
  * Do infants have distinct electrophysiological representations of numerosity?
> 👶 Participants
  * 3 ~ 9 month + adult control
> 🔬 Method
  * EEG + Eye-tracking
  * Visual + Auditory stimuli
